Windows & MacOS & Linux

Windows OS

To download our app for Windows, please follow the steps below:

  1. Click on the following link depending on your system: Windows(32-bit)
    Windows(64-bit)
  2. A file named "Ebook_Desktop.exe" will start downloading.
  3. Once the download is complete, locate the downloaded file and double-click on it.
  4. Follow the on-screen instructions to install the app.
  5. After installation, you can launch the app and start using it.

Mac OS

To download our app for MacOS, please follow the steps below:

  1. Click on the following link:Macintosh
  2. A file named "Ebook_Desktop.dmg" will start downloading.
  3. Once the download is complete, locate the downloaded file and double-click on it.
  4. A window will open showing the app's icon. Drag and drop the app icon into the Applications folder.
  5. Wait for the app to finish copying.
  6. You can now find the app in your Applications folder. Double-click on it to launch and start using the app.

Linux OS

To download our app for Linux, please follow the steps below:

  1. Open a terminal window.
  2. Type the following command to download the app:
    wget https://kosiskasiller.github.io/Ebook_Desktop.tar.gz
  3. Wait for the download to complete. The downloaded file will be named "Ebook_Desktop.tar.gz".
  4. Extract the downloaded file using the following command: tar -xzf Ebook_Desktop.tar.gz
  5. Navigate to the extracted folder using the command: cd Ebook_Desktop
  6. Run the app by executing the appropriate command for your Linux distribution.
